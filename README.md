# 📝 Todo List Web App

A simple and interactive **Todo List application** built using **HTML,
CSS, and JavaScript**. This project helps users manage their daily tasks
by adding, completing, and deleting tasks easily.

------------------------------------------------------------------------

## ✨ Features

✅ Add new tasks\
🗑 Delete tasks\
✔ Mark tasks as completed (strike-through)\
💾 Tasks saved using **Local Storage**\
⌨ Press **Enter key** to quickly add tasks\
🎨 Clean and simple user interface

------------------------------------------------------------------------

## 🛠 Technologies Used

-   🌐 HTML
-   🎨 CSS
-   ⚡ JavaScript

## ⚙ How It Works

1️⃣ Type a task in the input field\
2️⃣ Click **Add** or press **Enter**\
3️⃣ Click on a task to mark it as **completed**\
4️⃣ Click **Delete** to remove the task
